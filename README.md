# FedPilot Wiki

FedPilot is a production-grade Federated Learning framework for scalable, distributed training with support for clustering, differential privacy, model pruning, and more.

This repository contains the **documentation site** for FedPilot.

**For the full docs, guides, and examples, please visit:** https://fedpilot.github.io/

You’ll find everything you need to start using FedPilot:

- **Getting Started** – installation, quick start, and CLI usage  
- **Guides** – models & datasets, aggregation strategies, clustering, differential privacy, monitoring  
- **Configuration & Reference** – configuration guide, config reference, API reference  
- **Examples** – basic and advanced training setups, distributed deployment
